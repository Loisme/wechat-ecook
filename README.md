## 项目安装
``` bash
# 获取项目
git clone https://github.com/Loisme/wechat-ecook.git

# 进入文件夹
cd wechat-ecook

# 安装依赖(npm)
npm install

# 本地运行
npm run dev

# 项目发布
npm run build

# 生成 bundle 分析报告
npm run build --report
```
